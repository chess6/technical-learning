# Module validation record (Gate 10, module scope)

Fill-in template for a **module's benchmark validation** — the honest
covered / missing / deferred record for one module, measured against the course's
declared profile target. It consumes the
[module assessment plan](module-assessment-plan.md) results and feeds the
course-level Gate 10 report.

> **Saved as:** `docs/courses/<course>/modules/<module>/validation.md`.

Follow [authoring/mastery-standard.md §9 Gate 10](../mastery-standard.md#9-workflow-integration)
and the subject [benchmark matrix](../../<course>/benchmark-matrix.md) (link the real path).
